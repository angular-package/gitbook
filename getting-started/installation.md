# Installation

The package can be installed into your project via [NPM](https://www.npmjs.com), or [yarn](https://yarnpkg.com/getting-started/install).

{% tabs %}
{% tab title="npm" %}
```bash
npm install @angular-package/wrapper --save
```
{% endtab %}

{% tab title="yarn" %}
```
yarn add @angular-package/wrapper
```
{% endtab %}
{% endtabs %}
